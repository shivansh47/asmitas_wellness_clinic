# Asmita's Wellness Clinic

A nutrition management app that connects **dietitians** with their **clients** to help people achieve their health goals through personalized diet plans and meal tracking. An **admin** oversees the entire system.

## 🎯 What is Asmita's Wellness Clinic?

Asmita's Wellness Clinic is a digital platform where:
- **Clients** receive personalized diet plans from their dietitian and track their meals
- **Dietitians** create and manage nutrition plans for their clients and monitor their progress
- **Admins** manage the overall system and ensure everything runs smoothly

Think of it as a bridge between health professionals and people trying to improve their nutrition.

## ✨ What Makes It Special?

- **Works Everywhere**: One app that runs on phones (Android/iOS), tablets, computers (Windows/Mac/Linux), and web browsers
- **Real-Time Sync**: Changes appear instantly across all devices using cloud technology
- **Secure & Verified**: Uses Firebase's trusted authentication system to keep user data safe
- **Role-Based Access**: Each user type (Admin, Dietitian, Client) sees only what they need to see

## 🔧 Built With

**Technology Stack:**
- **Flutter + Dart** - The framework that allows the app to run on all devices from a single codebase
- **Firebase** - Cloud platform for user authentication and data storage
- **Provider** - State management system that keeps the app running smoothly

These technologies work together to create a fast, reliable app that works across all platforms.

## 🏗️ How the App is Organized

The project is divided into main sections:

- **Core** - The "engine" that handles user authentication, data fetching, and app logic
- **Screens** - The different pages users see (login page, admin dashboard, dietitian interface, client view, profile setup)
- **Widgets** - Reusable UI building blocks
- **Utils** - Helper functions and app styling (colors, fonts, design)
- **Firebase** - Connection settings to the cloud database

## 👥 Main Features

1. **User Authentication** - Secure login and registration for all three user types
2. **Personalized Dashboards** - Each user role gets a customized dashboard with relevant information
3. **Responsive Design** - The app looks great and works smoothly on any screen size (phone to desktop)
4. **Real-Time Updates** - Data syncs instantly across all devices when changes are made
5. **Multi-Platform Support** - Available on Android, iOS, Windows, Mac, Linux, and Web
